
Solution Insertion Sort:
Time: Best Case O(N), Average Case O(N^2), Worst Case O(N^2)
Space: O(1)

+

Solution Merge Sort:
Time: O(N log N) (Best Case, Worst Case, Average Case)
Space: O(N)

+

Solution Bubble Sort:
Time: O(N^2) (Best Case, Worst Case, Average Case)
Space: O(1)

+

Solution Quick Sort:
Time: Best Case O(N log N), Average Case O(N log N), Worst Case O(N^2)
Space: O(N)

+

Solution Selection Sort:
Time: O(N^2) (Best Case, Worst Case, Average Case)
Space: O(1)

+

Solution Binary Search:
Time: O(log N)
Space: O(1)

+

Solution Linear Search:
Time: Best Case O(1), Average Case O(N), Worst Case O(N)
Space: O(1)

+

Solution Summation of GCD of all the pairs up to N:
Time: O(MAX*log(log MAX))
Space: O(MAX)

+

Solution Pythagorean Quadruple:
Time: O(1) (Best Case, Worst Case, Average Case)
Space: O(1)
